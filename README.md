# IoT Spam Detection System

## Project Description
IoT Spam Detection System is a Django-based web application developed to detect and classify IoT messages as Spam or Normal messages.

## Features
- User Registration and Login
- IoT Message Prediction
- Spam Detection
- Admin Dashboard
- Prediction History
- MySQL Database Integration

## Technologies Used
- Python
- Django
- HTML
- CSS
- MySQL
- SQL

## Modules
- User Module
- Admin Module
- Prediction Module
- Dataset Management

## How to Run
1. Clone the repository
2. Install required packages
3. Run migrations
4. Start the Django server

```bash
python manage.py runserver
